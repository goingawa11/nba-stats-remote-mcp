1 Check Prerequisites

You need Node.js installed on your computer. This is required to connect Claude Desktop to the NBA Stats server.

How to check: Open Terminal (Mac) or Command Prompt (Windows) and type: node --version

If you see a version number (like v18.0.0), you're good! If not, download Node.js here (choose the LTS version).
2 Open Claude Desktop Settings
  1. Open Claude Desktop
  2. Click Claude in the menu bar (top left)
  3. Click Settings...
  4. Click Developer in the left sidebar
  5. Click Edit Config

This will open the config file in your default text editor.

~/Library/Application Support/Claude/claude_desktop_config.json
  1. Open Claude Desktop
  2. Click the ‚ò∞ menu icon (top left)
  3. Click Settings
  4. Click Developer in the left sidebar
  5. Click Edit Config

This will open the config file in Notepad.

%APPDATA%\Claude\claude_desktop_config.json
3 Add the NBA Stats Configuration

Replace the entire contents of the config file with this:

{ "mcpServers": { "nba-stats": { "command": "npx", "args": [ "mcp-remote", "https://nba-stats-remote-mcp-production.up.railway.app/mcp" ] } } }
Already have other MCP servers?

If your config file already has content, just add the "nba-stats": {...} section inside the existing "mcpServers" object. Make sure to add a comma after the previous server entry.

4 Restart Claude Desktop
  1. Save the config file and close it
  2. Completely quit Claude Desktop (not just close the window)
  3. Reopen Claude Desktop

The first time you use an NBA stats feature, it may take a few seconds to connect.

5 Try It Out!

Open a new conversation in Claude Desktop and try asking:

  • "What were the NBA scores last night?"
  • "How is LeBron James playing this season?"
  • "Who leads the league in assists?"
  • "Which team has the best defensive rating?"
Troubleshooting

Claude isn't using NBA stats: Make sure you restarted Claude Desktop completely. Try asking specifically: "Use the NBA stats tool to get today's scores."

"npx not found" error: Node.js isn't installed or not in your PATH. Reinstall Node.js from nodejs.org.

Connection errors: Check your internet connection. The server may occasionally be restarting - wait a minute and try again.
